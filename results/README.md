# Results

- `raw/`: large raw outputs from runs. Usually not committed.
- `processed/`: cleaned intermediate data. Usually not committed.
- `selected/`: small final result files for GitHub and the report.
- `figures/`: generated figures used in the report.
- `tables/`: generated LaTeX tables used in the report.
- `checkpoints/`: model checkpoints. Usually not committed.
- `logs/`: run logs. Usually not committed.
